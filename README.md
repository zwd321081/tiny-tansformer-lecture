# Tiny Transformer Learning

Goal: learn enough Python and PyTorch to hand-write a tiny Transformer language model.

Learning style:
- You write the core code.
- AI explains, gives hints, reviews, and helps debug.
- Progress is saved in files so a new session can continue from here.

## Roadmap

1. Python minimum basics
2. Tensor and shape basics
3. `nn.Embedding` and `nn.Linear`
4. Loss, `backward`, and optimizer
5. Bigram language model
6. Single-head causal self-attention
7. Tiny Transformer language model
